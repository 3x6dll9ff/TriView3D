# Глоссарий проекта

## Биологические термины

| Термин | Определение |
|--------|------------|
| **RBC** (Red Blood Cell) | Эритроцит — красная клетка крови |
| **Discocyte** | Нормальная двояковогнутая форма RBC (score = 0.00) |
| **Stomatocyte** | RBC с впадиной на одной стороне (score < 0) |
| **Echinocyte** | RBC с шиповидными выступами (score > 0) |
| **Spherocyte** | Сферическая RBC, патология (score = -1.00) |
| **iPSC** | Индуцированные плюрипотентные стволовые клетки |
| **z-стек** | Серия 2D-срезов объекта на разных глубинах фокуса |
| **Морфометрия** | Количественное измерение формы биологических структур |

## Технические термины

| Термин | Определение |
|--------|------------|
| **Voxel** | Объёмный пиксель — элемент 3D-сетки |
| **Ground truth** | Эталонные данные (в нашем случае — полные 3D-формы из `obj/`) |
| **Latent vector** | Сжатое внутреннее представление данных в автоэнкодере (256-dim) |
| **Sum projection** | Сумма вокселей вдоль одной оси → карта толщины с градиентом |
| **MIP** (Max Intensity Projection) | Максимум вдоль оси — теряет информацию о толщине |
| **BCE** (Binary Cross-Entropy) | Функция потерь для бинарной реконструкции |
| **Dice score** | Метрика пересечения predicted vs ground truth (2·|A∩B| / |A|+|B|) |
| **IoU** (Intersection over Union) | Метрика |A∩B| / |A∪B| |

## Метрики формы (morphometrics)

| Метрика | Формула/описание |
|---------|-----------------|
| **Volume** | Сумма ненулевых вокселей |
| **Sphericity** | Насколько форма близка к сфере (1.0 = идеальная сфера) |
| **Convexity** | Отношение объёма к выпуклой оболочке |
| **Surface roughness** | Шероховатость поверхности |
| **Eccentricity** | Степень вытянутости формы |
