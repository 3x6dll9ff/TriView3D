# Agent Skills

Навыки и исполняемые правила для AI-агента.

## Структура

Каждый скилл — отдельная подпапка с файлом `SKILL.md`:

| Папка | Описание |
|-------|----------|
| `frontend/SKILL.md` | Frontend — React, TypeScript, CSS, performance, accessibility |
| `backend/SKILL.md` | Backend — Python, API, безопасность, производительность |
| `engineering/SKILL.md` | Общие стандарты — Git, ADR, тесты, документация |
| `data-science/SKILL.md` | ML/DS — воспроизводимость, пайплайны, обучение, Colab |

## Как использовать

Перед задачей прочитать релевантный `SKILL.md` через `view_file`.
Маппинг задача → скилл описан в [[knowledge/rules]].
